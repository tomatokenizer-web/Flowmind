### Story 10.3: API Key Management

Add ApiKey model to prisma/schema.prisma:
model ApiKey {
  id        String   @id @default(cuid())
  userId    String   @map("user_id")
  name      String
  keyHash   String   @unique @map("key_hash")
  createdAt DateTime @default(now()) @map("created_at")
  lastUsed  DateTime? @map("last_used")
  user      User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  @@map("api_keys")
}
Add apiKeys ApiKey[] to User model.

Add apiKey tRPC router (src/server/api/routers/api-key.ts):
- apiKey.create: { name } - generates random key, stores hash, returns plain key once
