### Story 5.2: AI Text Decomposition (3-step process)
Add to src/server/ai/aiService.ts:
- `decomposeText(text, contextId, existingUnitIds)`:
  - Step 1: classify user purpose (arguing/brainstorming/researching/defining/other)
  - Step 2: propose decomposition boundaries (max 3 units per safety guard). Each boundary: startChar, endChar, content, proposedType, confidence (0-1)
  - Step 3: propose relations between new units AND existing units in context. Each: sourceIdx, targetUnitId, relationType, strength, rationale
  - If text < 20 chars: return single unit, no decomposition
  - All proposals have lifecycle='draft', originType='ai_generated'
  - Returns: { purpose, proposals: UnitProposal[], relationProposals: RelationProposal[] }
