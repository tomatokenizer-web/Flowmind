## Story 8.7: Unit Drift Detection

Create src/server/services/driftService.ts:
- computeDriftScore(unitContent, projectPurpose): simple keyword overlap score (0-1)
  - If no projectPurpose: return 0 (no drift possible)
  - Count keyword matches between unit content and project purpose
  - Score = 1 - (matches / totalKeywords)
- updateDriftScoresForProject(projectId, db): batch update all units' drift_score
- getHighDriftUnits(projectId, threshold, db): returns units with drift_score > threshold

Add project.getDriftUnits tRPC procedure: { projectId, threshold?: number }
Add project.resolveDrift tRPC procedure: { unitId, action: 'keep'|'move'|'branch' }
