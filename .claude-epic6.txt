## Story 6.4: Multi-Layer Search

Create src/server/services/searchService.ts:
- search(query, options: { contextId?, layers: ('text'|'semantic'|'structural'|'temporal')[] }):
  - text: PostgreSQL full-text search on unit content (use Prisma raw query with to_tsquery)
  - semantic: vector similarity search using pgvector (cosine similarity on embedding column, if embedding exists)
  - structural: search by unit type, lifecycle, relation count
  - temporal: filter by date ranges

## Story 6.5: ThoughtRank Importance Score

Create src/server/services/thoughtRankService.ts:
- computeThoughtRank(units, relations): PageRank-inspired algorithm
  - Each unit starts with rank 1/N
  - Iteratively distribute rank through outgoing relations weighted by strength
  - Damping factor 0.85, 20 iterations
  - Returns: Map<unitId, score>
- updateThoughtRankForContext(contextId): runs computation and updates unit.importance in DB

Add context.recomputeThoughtRank tRPC procedure.
Trigger ThoughtRank update on relation.create/update/delete (background, non-blocking).
Display importance score subtly on UnitCard (small bar or number).
